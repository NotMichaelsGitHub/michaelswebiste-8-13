Projects

Pokemon TCG Starting Hand Simulator

This web app allows you to import a PTCGO decklist paste, and simulates your starting options

It's useful tool to use when deck building to help lay out what consistency issues you could potentially face

Is it Rotating? Pokemon TCG 2019-2020 season

Have a hard time remember what Pokemon cards came out in what set when planning for post rotation?

I created a web app to help out with just that. This tool allows you to search for a card from the Sun and Moon Series and lets you know if it's rotating
